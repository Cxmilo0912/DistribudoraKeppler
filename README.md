# DistribudoraKeppler
-Este proyecto consiste en el desarrollo de un sistema web Multiplataforma para la gestión de una distribuidora, permitiendo administrar productos, inventario, ventas y usuarios (funcionarios y clientes).

-El sistema busca optimizar los procesos comerciales, mejorar el control del stock y facilitar la toma de decisiones mediante información organizada.
## Tecnologias Utilizadas
- Backend: ASP.NET (C#)
- Frontend: HTML, CSS, Bootstrap
- Base de datos: SQL Server
- Control de versiones: Git y GitHub

# Instrucciones de Ejecucion del Proyecto
## Requisitos previos
- Tener instalado Visual Studio
- Tener SQL Server configurado
- Tener Git instalado
## Instalación
1. git clone https://github.com/Cxmilo0912/DistribuidoraKeppler.git
2. cd DistribuidoraKeppler
## Ejecución
Abrir el proyecto en Visual Studio
Configurar la cadena de conexión a la base de datos
Ejecutar el proyecto (F5 o botón "Iniciar")
